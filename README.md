# 聽雨

> 雨聲細碎，字句留痕。

**聽雨** — 個人部落格，記錄閱讀、思考與生活的點滴。

🌐 **網站：** https://tingyu.wahsun.org

---

## 關於

聽雨是一個以文字為核心的個人部落格，專注於書評、思想隨筆與生活記錄。
設計風格取法水墨意境，以簡潔的排版與溫潤的色調，讓閱讀回歸內容本身。

## 技術棧

| 項目 | 採用 |
|------|------|
| 框架 | [Astro](https://astro.build) 6.x |
| 主題 | Tingyu,完全由WAHSUN設計製作 |
| 部署 | [Cloudflare Pages](https://pages.cloudflare.com) |
| 網域 | wahsun.org (Cloudflare DNS) |

## 開發

```bash
npm install
npm run dev      # 本地開發
npm run build    # 生產構建
```
